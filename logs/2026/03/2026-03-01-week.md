## C++

**环境与开发基础**  

有待学习
- `Ubuntu 22.04`（或 `WSL2 + Ubuntu`）作为开发终端环境
- `Git + GitHub` 做版本管理与远端仓库协作
- `README/.gitignore/.env.example` 做工程规范与可复现配置

**容器与部署编排**

- `Docker` 做服务容器化
- `Docker Compose` 做多服务编排（四服务一起启动）
- `Dockerfile` 分别构建 `cpp-gateway` 与 `python-ai`
- `healthcheck + depends_on` 做服务健康探测与启动顺序控制

**后端服务技术**

- `C++17 + Drogon + CMake`：网关服务（REST API，当前实现 `/health`）
- `Python 3.11 + FastAPI + Uvicorn`：AI 占位服务（当前实现 `/health`）
- `HTTP/JSON`：服务对外接口协议与返回格式

**数据与缓存层**

- `PostgreSQL 16`
- `pgvector` 扩展（向量字段：`vector(1536)`）
- `Redis 7`（缓存与后续限流/任务状态基础）
- `SQL DDL + Index`：`docs/chunks/tasks/rag_logs` 表及基础索引

**验证与运维操作**

- `curl` 做接口健康验证
- `docker compose ps/logs` 做服务状态与日志排查
- `docker compose up --build -d` 完成一键启动

**一句话技术栈**

- 基于 `C++ Drogon` 的 API Gateway + `Python FastAPI` AI 服务，使用 `Docker Compose` 编排，底层采用 `PostgreSQL(pgvector) + Redis` 的 RAG 工程骨架。
